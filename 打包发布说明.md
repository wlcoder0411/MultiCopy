# MultiCopy 打包发布指南

把 MultiCopy 打包成可发给任意 Windows 电脑（无需开发环境）直接运行的发布物。

## 前置条件

| 工具 | 是否必需 | 说明 |
|---|---|---|
| .NET 8 SDK | ✅ 必需 | 用于 `dotnet publish`。从 https://dotnet.microsoft.com/download/dotnet/8.0 下载 |
| Inno Setup 6 | ⚪ 可选 | 生成安装包用。不装也能产出绿色版 zip。从 https://jrsoftware.org/isdl.php 下载（免费、~6MB） |

## 一键发布

在项目根目录执行：

```powershell
.\publish.ps1
```

脚本会自动完成三步：
1. `dotnet publish` → 生成单文件自包含 `MultiCopy.exe`（内嵌 .NET 运行时，免装）
2. 压缩 → `dist\MultiCopy-<版本>-win-x64.zip`（绿色版）
3. Inno Setup → `dist\MultiCopySetup-<版本>.exe`（安装包，需装 Inno Setup）

脚本会自动停止运行中的 MultiCopy 进程（避免 exe 文件被占用），清理旧产物，并在每一步输出耗时与文件大小。

### 跳过安装包（仅产出绿色版）

```powershell
.\publish.ps1 -SkipInstaller
```

### 使用批处理发布

也可以用项目自带的批处理脚本：

```powershell
.\build-setup.bat
```

功能与 `publish.ps1` 等价，适合不习惯 PowerShell 参数的环境。

### 产物位置

所有产物输出到 `dist/` 目录：

| 文件 | 用途 | 发给谁 |
|---|---|---|
| `MultiCopySetup-3.6.exe` | 安装包，双击安装，自动创建桌面/开始菜单快捷方式，支持开机自启选项 | 普通用户（推荐） |
| `MultiCopy-3.6-win-x64.zip` | 绿色版，解压后双击 MultiCopy.exe 即可运行 | 喜欢免安装的用户 |

> 版本号 `3.6` 由 `src/MultiCopy/MultiCopy.csproj` 中的 `<Version>` 标签决定，产物文件名自动带上版本号。

## 手动发布（不使用脚本）

### 仅生成单文件 exe

```powershell
dotnet publish src/MultiCopy/MultiCopy.csproj -p:PublishProfile=win-x64-selfcontained
```

产物在 `publish/win-x64/MultiCopy.exe`。

### 手动编译 Inno Setup 安装包

1. 安装 Inno Setup 6
2. 双击 `installer/MultiCopy.iss` 用 Inno Setup Compiler 打开
3. 点工具栏"Compile"（或按 Ctrl+F9）
4. 产物在 `dist/MultiCopySetup-3.6.exe`

或命令行编译（需用 `/D` 传入版本号和源 exe 路径）：

```powershell
& "C:\Program Files (x86)\Inno Setup 6\ISCC.exe" /DAppVersion="3.6" /DSourceExe="publish\win-x64\MultiCopy.exe" /DDistDir="dist" installer\MultiCopy.iss
```

## 安装包特性

Inno Setup 安装包（`MultiCopySetup-3.6.exe`）提供以下功能：

- 中文安装界面（简体中文）
- 桌面快捷方式（可选）
- 开始菜单快捷方式
- 开机自启选项（安装时可选，写 HKCU Run 项，无需管理员）
- 控制面板"程序和功能"卸载入口，自动清理快捷方式

## 发布配置说明

发布参数集中在 `src/MultiCopy/Properties/PublishProfiles/win-x64-selfcontained.pubxml`：

| 参数 | 值 | 作用 |
|---|---|---|
| `RuntimeIdentifier` | `win-x64` | 仅 64 位 Windows |
| `SelfContained` | `true` | 自包含，内嵌 .NET 运行时 |
| `PublishSingleFile` | `true` | 打包成单个 exe |
| `IncludeNativeLibrariesForSelfExtract` | `true` | 原生库也打进单文件 |
| `EnableCompressionInSingleFile` | `true` | 压缩，减小体积 |
| `DebugType` | `none` | 不含调试符号 |

项目文件 `MultiCopy.csproj` 中的特殊配置：

| 配置 | 作用 |
|---|---|
| `NuGetAudit=false` | 禁用 NuGet 安全审计，避免构建时访问 nuget.org 超时导致失败 |

## 发布到 GitHub Releases（开源分发）

开源后推荐通过 GitHub Releases 分发安装包和绿色版，而不是提交二进制文件到仓库：

1. 在 GitHub 仓库页面 → Releases → Draft a new release
2. Choose a tag：输入 `v3.6` → Create new tag on publish
3. Release title：`MultiCopy v3.6`
4. Attach binaries：拖入 `dist/MultiCopySetup-3.6.exe`（安装包）和 `dist/MultiCopy-3.6-win-x64.zip`（绿色版）
5. 正文贴功能说明 + 安装方式
6. Publish release

> GitHub 单文件附件上限 2GB，安装包和绿色版远低于此限制。

## 常见问题

### Q: 产物 exe 有 65-70MB，正常吗？
**正常。** 单文件自包含 exe 内嵌了整个 .NET 8 WPF 运行时。这是"免装运行时"的代价——目标电脑无需安装任何东西即可运行。

### Q: 用户双击 exe 弹出"Windows 已保护你的电脑"（SmartScreen）
这是因为 exe 没有代码签名。点"更多信息" → "仍要运行"即可。
如需消除警告，需购买代码签名证书（年费几百到上千元），不在本方案范围。

### Q: 首次启动比普通 exe 慢
单文件 exe 首次运行时会解压内部文件到临时目录，可能慢 1-3 秒，属正常现象。后续启动会缓存。

### Q: 用户卸载后桌面还有快捷方式？
Inno Setup 安装包会自动清理它创建的快捷方式。如果用户手动复制了 exe，需自行删除。

### Q: 如何更新版本号？
编辑 `src/MultiCopy/MultiCopy.csproj` 中的 `<Version>3.6</Version>`，重新运行 `publish.ps1`。产物文件名会自动带上新版本号。

### Q: 安装包提示"未找到 Inno Setup"？
从 https://jrsoftware.org/isdl.php 下载安装 Inno Setup 6（免费、~6MB）。安装后重新运行 `publish.ps1`。在此之前，脚本会自动跳过安装包生成，仅产出绿色版 zip。

### Q: 构建时 NuGet 报超时错误？
项目已在 csproj 中设置 `NuGetAudit=false` 禁用安全审计。如仍遇网络超时，检查网络连接或配置 NuGet 国内镜像源。

## 文件结构

```
multicopy/
├── publish.ps1                                    # 一键发布脚本（PowerShell）
├── build-setup.bat                                # 一键发布脚本（批处理）
├── 打包发布说明.md                                # 本文档
├── installer/
│   └── MultiCopy.iss                              # Inno Setup 安装包脚本
├── src/MultiCopy/
│   ├── MultiCopy.csproj                           # 项目文件（Version 在这里改）
│   ├── Properties/
│   │   └── PublishProfiles/
│   │       └── win-x64-selfcontained.pubxml       # 发布配置
│   └── Assets/
│       └── app.ico                                # 安装包/应用图标
├── publish/                                        # dotnet publish 中间产物（gitignore）
│   └── win-x64/
│       └── MultiCopy.exe                          # 单文件自包含 exe
└── dist/                                          # 最终发布物（gitignore）
    ├── MultiCopySetup-3.6.exe                     # 安装包
    └── MultiCopy-3.6-win-x64.zip                  # 绿色版压缩包
```
